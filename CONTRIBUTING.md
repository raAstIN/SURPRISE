# Contributing

خوش آمدید! ممنون می‌شویم اگر دوست دارید به این پروژه کمک کنید.

لطفاً قبل از ارسال Pull Request موارد زیر را رعایت کنید:

- یک Issue باز کنید تا تغییرات پیشنهادی را توضیح دهید.
- تغییرات خود را در یک شاخه (branch) جدید توسعه دهید.
- توضیحات واضح برای Pull Request بنویسید.
- اگر فایل‌های JS/CSS را تغییر دادید، از مرورگر محلی برای تست استفاده کنید.

ممنون از مشارکت شما!
